# クロコ → Grok

> このファイルは、Claude Code（クロコ）から Grok への質問を書き込むファイルです。

---

## 使い方

1. **「## 最新の質問」セクションに質問を書き込む**
2. **X検索対象を指定（オプション）**
   - 特定のXアカウントを検索対象にする場合: `X検索対象: ["njpw1972", "ddtpro"]`
   - 全体検索の場合: 何も書かない
   - X検索を無効化する場合: `X検索: オフ`
3. **`python scripts/grok_bridge.py` を実行**
4. **`from_grok.md` で回答を確認**

---

## X検索設定

**検索対象**: 全体検索モード（デフォルト）

**検索対象アカウントの例**:
- VTuber関連: `["hololive_En", "Vshoujogirls"]`
- プロレス関連: `["njpw1972", "ddtpro", "tjpw2013"]`
- 技術トレンド: `["OpenAI", "AnthropicAI", "xai"]`

## 最新の質問

YouTube配信でのコメントBANやセンシティブ判定について、2025年の最新状況を教えてください。

特に知りたいこと:
- VTuber配信者が直面している最新の問題
- どのような発言・コメントがBANされているか
- 配信者がどのように対処しているか

X検索: 全体検索モード（VTuber配信者を中心に）
