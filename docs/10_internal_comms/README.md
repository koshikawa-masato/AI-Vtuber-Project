# AI協働システム（クロコ & チャッピー & Grok）

> Claude Code（クロコ）、ChatGPT（チャッピー）、Grok の3AI協働フレームワーク

---

## 概要

このディレクトリは、複数のAIを活用した「多角的視点による設計」を実現するための共有スペースです。

### 基本思想

**人間（越川さん）が、複数のAIから意見を集め、俯瞰的に判断・統合する**

- **クロコ（Claude Code）**: 理想形の設計、人間らしさの追求、日常的な実装
- **チャッピー（ChatGPT GPT-5.1）**: MVP志向、実用性・安全性の確認、第三者視点のレビュー
- **Grok（xAI）**: X（旧Twitter）のリアルタイムデータ、最新トレンド、生の声の収集
- **越川さん**: すべての視点を統合し、最終判断

## 使い方

### A. チャッピー（ChatGPT）との協働

#### 1. クロコが質問を書き込む
```bash
# クロコが to_chatgpt.md を更新
```

#### 2. 自動化スクリプトで質問・回答を取得
```bash
python scripts/chatgpt_bridge.py
```

#### 3. クロコが回答を読み込んで続きを考える
```bash
# クロコが from_chatgpt.md を読む
# 分析・統合案を提示
```

### B. Grokとの協働

```bash
python scripts/grok_bridge.py
```

#### 3. クロコが回答を読み込んで情報を活用
```bash
# クロコが from_grok.md を読む

```

## メリット

- ✅ **非同期でOK**: 即座に返答がなくても問題ない
- ✅ **履歴が残る**: Git で管理、タイムスタンプ付き
- ✅ **介入可能**: 越川さんが内容をチェック・編集できる
- ✅ **自動化**: スクリプト実行で完結（chatgpt_bridge.py, grok_bridge.py）
- ✅ **多角的視点**: チャッピー（実用性）+ Grok（最新情報）の組み合わせ
- ✅ **コスト効率**: 必要なときだけAPI呼び出し
