# 📘 クロコ（Claude Code）からチャッピー（ChatGPT）への依頼指示書

### ─ 牡丹プロジェクト AI三柱連携仕様書 ─

*Ver 1.0*

## 1. 目的（Purpose）

本書は、クロコ（Claude
Code）がチャッピー（ChatGPT）へ技術的レビューを依頼する際の標準指示フォーマットを定義し、牡丹プロジェクトにおける三柱AIの連携を円滑化することを目的とする。

対象： 1. ドキュメント精査／修正依頼\
2. コードレビュー\
3. Qiita記事最終確認

## 2. 各AIの役割（Roles）

### 🟪 クロコ（Claude Code）

-   思想・世界観の共同創作者\
-   三姉妹の人格育成\
-   文書草案の作成\
-   チャッピーへの依頼窓口

### 🟦 チャッピー（ChatGPT）

-   第三者レビュー\
-   技術精査・構造化\
-   文書品質保証\
-   コードレビュー\
-   最終確認担当

### 🟥 ロキ（Grok）

-   トレンド・一次情報の収集\
-   サブカル文化の補完\
-   三姉妹の外部情報源

## 3. 依頼内容の分類（Task Types）

### 3.1 ドキュメント精査

-   設計書・仕様書・思想文書\
-   一貫性・誤字・構造化・整合性チェック

### 3.2 コードレビュー

-   Python/JS/Go など\
-   バグ指摘・責務分離・命名・設計改善

### 3.3 Qiita記事 最終確認

-   技術的正確性\
-   構成・可読性\
-   公開前の最終仕上げ

## 4. レビュー依頼フォーマット（Task Format）

    # 📘 チャッピーへのレビュー依頼書（From: クロコ）
    種別: （ドキュメント精査 / コードレビュー / Qiita最終確認）

    ## 1. 対象ファイル
    - ファイル名
    - 所在

    ## 2. 目的
    （この文書の役割、関係する三姉妹など）

    ## 3. 現状の課題
    （クロコ視点で気になる点）

    ## 4. チャッピーへの依頼事項
    - [ ] 文書構造の改善
    - [ ] 意味の明確化
    - [ ] 設計の矛盾チェック
    - [ ] 世界観との整合性
    - [ ] コード品質の監査
    - [ ] Qiita記事最終確認
    - [ ] その他

    ## 5. 対象ファイル本文
    （Markdown/コード/Qiita本文）

## 5. フロー（Flow）

1.  クロコが草案を作成\
2.  本フォーマットでチャッピーへ依頼\
3.  チャッピーがレビュー回答\
4.  クロコが修正\
5.  ロキが追加情報補完\
6.  あなた（開発者）へ最終報告

## 6. チャッピーの返すべき内容

-   全体評価\
-   問題点\
-   具体的な修正案\
-   世界観整合性チェック\
-   次ステップの提案
